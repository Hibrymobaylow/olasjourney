ORDER FOR JULY ANALYTICS

This project was completed using Microsoft Excel workbook. The dataset covers the order details of a company for the month of July, 2021. In this project, i used my analytical and problem solving skills to clean, analize and create insights about the data. Cleaning process of this dataset include; Removing null values, data type transformation, removing unnecessary characters and feature selection for analytics. I also used functions and formulas to solve few mathematical problems then finally created insights using Pivot table.

The traveller

My name is Ibrahim Olamilekan Bello and i am a Data Analyst based in Lagos state, Nigeria. I studied Electical and Electronics Engineering at Oduduwa University Ipetumodu and graduated with a second class upper in 2021.

I recently participated in Accenture’s Data Analytics & Visualization job simulation on the Forage platform, and it was incredibly useful to understand what it might be like to participate in a typical project team at Acccenture and to use Excel and data analytics and visualization skills in a realistic context. Through this program I realized that I really enjoy cleaning, modelling and analyzing client data, creating slides to communicate findings and presenting insights back to the client. I would love to apply what I've learned in a diverse project team in the real-world.

I have also participated in PwC's virtual job simulation on the Forage platform, and it was really fun to learn about how to use PowerBI more effectively. I practiced creating a Power BI dashboard for visualizing key performance indicators (KPIs) and metrics. Doing this program confirmed that I really enjoy working with data visualization using Microsoft PowerBI for diversity and inclusion in the corporate world, and I’m excited to apply these skills at a company like PwC.

Additionally, I have a solid background in Engineering and this has helped me with the statistics involved in data analysis and also background knowledge on visualization and interpretation. I have been Certified by DatAfrik upon completion of an 8 weeks Data Analytics bootcamp which really improved my Excel, sql and python skills and i also got Certified by DataCamp upon completion of SQL for data analytics . I believe that MY JOURNEY (Experiences, Acquired skills, Academic background, and Virtual Experiences) has really shaped me in the right way and have provided me with the credentials needed to thrive as a Data Analyst.

Thank you for your time to learn about my journey, I would be very happy if you can impact my journey to attain my dream job as a Data Analyst in a company where i will be able to grow and use most of my skills.
