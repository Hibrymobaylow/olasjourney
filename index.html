ORDER FOR JULY ANALYTICS

This project was completed using Microsoft Excel workbook. The dataset covers the order details of a company for the month of July, 2021. This was a project containing order details for a company in the month of July 2021. Through the Microsoft Excel workbook, the project sought to use my analytical and problem-solving skills in conducting a detailed cleaning exercise on the data in presentation of important actionable insights. Among them was the removal of null values, changing the data types, trimming off characters which were irrelevant, and then selecting features to be used in analysis. I applied several functions and formulae in Excel to solve the mathematical problems within the dataset. Finally, I used Pivot Tables to generate detailed insights that would give a clear overview of the order metrics and performance of the company for the month.

Visualization

In this project, I gathered major insights and KPIs and developed a comprehensive, interactive dashboard using Power BI. Before making the dashboards, I had used Power Query to preprocess the data by scrubbing and transforming it very carefully to make sure that the insights developed from it were correct and reliable. Handling missing values, correct data types, and data normalization for consistency were part of this process. Finally, the dashboards portray the data in a clear, active, and insightful way for visualization that will support an informed decision-making process.
